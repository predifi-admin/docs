## Market Data

Fetch market data, orderbooks, and historical prices.

---

## List All Markets

```typescript
const markets = await client.markets.list({
  category: 'crypto', // or 'sports', 'politics'
  status: 'active' // or 'resolved', 'all'
});
```

---

## Get Market Details

```typescript
const market = await client.markets.get('btc-100k-jan15');

console.log({
  yesPrice: market.yesPrice,
  noPrice: market.noPrice,
  volume24h: market.volume24h,
  liquidity: market.liquidity
});
```

---

## Get Orderbook

```typescript
const orderbook = await client.markets.getOrderbook('btc-100k-jan15');

console.log({
  bids: orderbook.bids, // [{ price: 0.64, size: 100 }, ...]
  asks: orderbook.asks  // [{ price: 0.66, size: 150 }, ...]
});
```

---

## WebSocket Real-Time Data

```typescript
const ws = client.websocket.connect();

ws.subscribe('market:btc-100k-jan15', (data) => {
  console.log('Price update:', data.yesPrice, data.noPrice);
});
```
