## Official SDKs

Predifi provides official SDKs for popular programming languages.

---

## TypeScript / JavaScript

```bash
npm install @predifi/sdk
```

**GitHub:** [github.com/predifi-com/predifi-ts](https://github.com/predifi-com/predifi-ts)

**Example:**
```typescript
import { PredifiClient } from '@predifi/sdk';

const client = new PredifiClient({ apiKey: 'your-key' });
const markets = await client.markets.list();
```

---

## Python

```bash
pip install predifi-python
```

**GitHub:** [github.com/predifi-com/predifi-python](https://github.com/predifi-com/predifi-python)

**Example:**
```python
from predifi import PredifiClient

client = PredifiClient(api_key='your-key')
markets = client.markets.list()
```

## Features

All SDKs support:

- ✅ Market data fetching
- ✅ Order placement (market & limit)
- ✅ Portfolio tracking
- ✅ WebSocket subscriptions
- ✅ Type-safe interfaces
- ✅ Comprehensive error handling
