## Overview

The Markets API provides access to:
- **Market discovery** - Browse and search all markets
- **Market details** - Full metadata and trading information
- **Price history** - Historical price data with intervals
- **Recent trades** - Trade execution stream

---

## List Markets

Get all markets with filtering and sorting.

<ParamField path="GET" type="method">
  /v1/markets
</ParamField>

### Parameters

<ParamField query="category" type="string">
  Filter by category: `crypto`, `sports`, `politics`, `finance`, `other`
</ParamField>

<ParamField query="status" type="string">
  Filter by status: `active`, `closed`, `resolved`
</ParamField>

<ParamField query="asset" type="string">
  Filter by underlying asset (e.g., `BTC`, `ETH`, `SPX`)
</ParamField>

<ParamField query="interval" type="string">
  Filter by time interval: `15M`, `1H`, `4H`, `1D`, `1W`
</ParamField>

<ParamField query="sort" type="string">
  Sort by: `volume`, `liquidity`, `created`, `closing` (default: `volume`)
</ParamField>

<ParamField query="order" type="string">
  Sort order: `asc`, `desc` (default: `desc`)
</ParamField>

<ParamField query="page" type="number">
  Page number (default: 1)
</ParamField>

<ParamField query="limit" type="number">
  Results per page (max: 100, default: 20)
</ParamField>

### Response

### Example

<CodeGroup>

```bash cURL
curl "https://api.predifi.com/v1/markets?category=crypto&status=active&sort=volume&limit=10" \
  -H "Authorization: Bearer YOUR_API_KEY"
```

```typescript TypeScript
const markets = await api.get('/v1/markets', {
  params: {
    category: 'crypto',
    status: 'active',
    sort: 'volume',
    limit: 10
  }
});

console.log(`Found ${markets.data.length} markets`);
markets.data.forEach(m => {
  console.log(`${m.question}`);
  console.log(`  YES: $${m.currentPrice.yes} | Volume: $${m.volume24h}`);
});
```

```python Python
response = requests.get(
    'https://api.predifi.com/v1/markets',
    headers={'Authorization': f'Bearer {api_key}'},
    params={'category': 'crypto', 'status': 'active', 'limit': 10}
)

for market in response.json()['data']:
    print(f"{market['question']}: ${market['currentPrice']['yes']}")
```

</CodeGroup>

## Get Market Details

Get full details for a specific market.

<ParamField path="GET" type="method">
  /v1/markets/:id
</ParamField>

### Response

Includes all fields from list endpoint plus:

<ResponseField name="description" type="string">
  Detailed market description and rules
</ResponseField>

<ResponseField name="volumeTotal" type="number">
  Total lifetime volume
</ResponseField>

<ResponseField name="metadata" type="object">
  Additional market metadata (creator, tags, etc.)
</ResponseField>

### Example

<CodeGroup>

```bash cURL
curl "https://api.predifi.com/v1/markets/BTC-15M-20250111-1430" \
  -H "Authorization: Bearer YOUR_API_KEY"
```

```typescript TypeScript
const market = await api.get('/v1/markets/BTC-15M-20250111-1430');
console.log(market.data.question);
console.log(market.data.description);
console.log(`Total volume: $${market.data.volumeTotal}`);
```

```python Python
response = requests.get(
    'https://api.predifi.com/v1/markets/BTC-15M-20250111-1430',
    headers={'Authorization': f'Bearer {api_key}'}
)
market = response.json()['data']
print(f"Volume: ${market['volumeTotal']}")
```

</CodeGroup>

## Search Markets

Full-text search across market questions and descriptions.

<ParamField path="GET" type="method">
  /v1/markets/search
</ParamField>

### Parameters

<ParamField query="q" type="string" required>
  Search query
</ParamField>

<ParamField query="category" type="string">
  Filter by category
</ParamField>

<ParamField query="status" type="string">
  Filter by status
</ParamField>

### Response

Same format as list markets.

### Example

<CodeGroup>

```typescript TypeScript
const results = await api.get('/v1/markets/search', {
  params: { q: 'bitcoin 100k', category: 'crypto' }
});

console.log(`Found ${results.data.length} markets matching "bitcoin 100k"`);
```

```python Python
response = requests.get(
    'https://api.predifi.com/v1/markets/search',
    headers={'Authorization': f'Bearer {api_key}'},
    params={'q': 'bitcoin 100k'}
)
```

</CodeGroup>

## Get Price History

Historical price data with configurable intervals.

<ParamField path="GET" type="method">
  /v1/markets/:id/prices
</ParamField>

### Parameters

<ParamField query="interval" type="string">
  Interval: `1m`, `5m`, `15m`, `1h`, `4h`, `1d` (default: `1h`)
</ParamField>

<ParamField query="from" type="number">
  Start timestamp (Unix ms)
</ParamField>

<ParamField query="to" type="number">
  End timestamp (Unix ms)
</ParamField>

### Response

```json
{
  "ok": true,
  "data": {
    "marketId": "BTC-15M-20250111-1430",
    "interval": "1h",
    "prices": [
      {
        "timestamp": 1638356400000,
        "yesPrice": 0.6234,
        "noPrice": 0.3766,
        "volume": 12345.67,
        "liquidity": 87654.32
      },
      {
        "timestamp": 1638360000000,
        "yesPrice": 0.6543,
        "noPrice": 0.3457,
        "volume": 23456.78,
        "liquidity": 89012.34
      }
    ]
  }
}
```

### Example

<CodeGroup>

```typescript TypeScript
const history = await api.get('/v1/markets/BTC-15M-20250111-1430/prices', {
  params: { interval: '15m' }
});

history.data.prices.forEach(p => {
  const date = new Date(p.timestamp);
  console.log(`${date.toLocaleString()}: $${p.yesPrice}`);
});
```

```python Python
response = requests.get(
    'https://api.predifi.com/v1/markets/BTC-15M-20250111-1430/prices',
    headers={'Authorization': f'Bearer {api_key}'},
    params={'interval': '15m'}
)

for price in response.json()['data']['prices']:
    print(f"Time: {price['timestamp']}, Price: ${price['yesPrice']}")
```

</CodeGroup>

## Get Recent Trades

Get recent trade executions for a market.

<ParamField path="GET" type="method">
  /v1/markets/:id/trades
</ParamField>

### Parameters

<ParamField query="limit" type="number">
  Number of trades (max: 100, default: 20)
</ParamField>

### Response

```json
{
  "ok": true,
  "data": [
    {
      "tradeId": "0x123...",
      "marketId": "BTC-15M-20250111-1430",
      "outcomeId": 0,
      "side": "buy",
      "price": 0.6543,
      "size": 1000.00,
      "value": 654.30,
      "venue": "polymarket",
      "timestamp": 1638360000000
    }
  ]
}
```

### Example

<CodeGroup>

```typescript TypeScript
const trades = await api.get('/v1/markets/BTC-15M-20250111-1430/trades', {
  params: { limit: 50 }
});

trades.data.forEach(t => {
  console.log(`${t.side.toUpperCase()} ${t.size} @ $${t.price} on ${t.venue}`);
});
```

```python Python
response = requests.get(
    'https://api.predifi.com/v1/markets/BTC-15M-20250111-1430/trades',
    headers={'Authorization': f'Bearer {api_key}'},
    params={'limit': 50}
)

for trade in response.json()['data']:
    print(f"{trade['side']} {trade['size']} @ ${trade['price']}")
```

</CodeGroup>

## Response Fields

<ResponseField name="marketId" type="string">
  Unique market identifier
</ResponseField>

<ResponseField name="question" type="string">
  Market question
</ResponseField>

<ResponseField name="category" type="string">
  Market category
</ResponseField>

<ResponseField name="status" type="string">
  `active`, `closed`, or `resolved`
</ResponseField>

<ResponseField name="currentPrice" type="object">
  Current YES/NO prices
</ResponseField>

<ResponseField name="volume24h" type="number">
  24-hour trading volume in USDC
</ResponseField>

<ResponseField name="liquidity" type="number">
  Available liquidity in USDC
</ResponseField>

<ResponseField name="venues" type="object">
  Per-venue breakdown of volume and liquidity
</ResponseField>
