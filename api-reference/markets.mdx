## Overview

The Markets API enables you to:
- **List all markets** - Browse active, upcoming, and closed markets
- **Search markets** - Find markets by keywords
- **Filter by category** - Crypto, sports, and more
- **Get market details** - Full market information including pricing
- **Historical data** - Price history and trading volume

---

## List Markets

Get all available prediction markets.

<ParamField path="GET" type="method">
  /v1/markets
</ParamField>

### Request

<ParamField query="category" type="string">
  Filter by category:
  - `crypto` - Cryptocurrency markets
  - `sports` - Sports betting markets
  - `all` - All categories (default)
</ParamField>

<ParamField query="status" type="string">
  Filter by market status:
  - `active` - Currently trading (default)
  - `upcoming` - Not yet open
  - `closed` - Ended, awaiting settlement
  - `settled` - Fully resolved
  - `all` - All statuses
</ParamField>

<ParamField query="asset" type="string">
  Filter by underlying asset (crypto only):
  - `BTC` - Bitcoin markets
  - `ETH` - Ethereum markets
  - `SOL` - Solana markets
  - `BNB` - Binance Coin markets
  - `DOGE` - Dogecoin markets
  - `XRP` - Ripple markets
</ParamField>

<ParamField query="interval" type="string">
  Filter by time interval (crypto only):
  - `15M` - 15-minute markets
  - `30M` - 30-minute markets
  - `1H` - 1-hour markets
</ParamField>

<ParamField query="sort" type="string">
  Sort results by:
  - `volume` - Trading volume (default)
  - `liquidity` - Total liquidity
  - `expiry` - Expiration time
  - `created` - Creation time
</ParamField>

<ParamField query="order" type="string">
  Sort order:
  - `desc` - Descending (default)
  - `asc` - Ascending
</ParamField>

<ParamField query="page" type="number">
  Page number (default: 1)
</ParamField>

<ParamField query="limit" type="number">
  Results per page (default: 20, max: 100)
</ParamField>

<ResponseField name="marketId" type="string">
  Unique market identifier
  
  Format: `{ASSET}-{INTERVAL}-{DATE}-{TIME}`
</ResponseField>

<ResponseField name="question" type="string">
  Market question/description
</ResponseField>

<ResponseField name="category" type="string">
  Market category: `crypto`, `sports`, etc.
</ResponseField>

<ResponseField name="status" type="string">
  Market status:
  - `active` - Currently trading
  - `upcoming` - Not yet open (created but trading not started)
  - `closed` - Expired, awaiting settlement
  - `settled` - Fully resolved
  - `cancelled` - Market cancelled (rare)
</ResponseField>

<ResponseField name="expiresAt" type="number">
  Market expiration timestamp (Unix ms)
  
  When trading stops and settlement begins
</ResponseField>

<ResponseField name="timeRemaining" type="number">
  Milliseconds until expiration
</ResponseField>

<ResponseField name="currentPrice" type="object">
  Current YES/NO prices (0-1)
  
  Updated in real-time
</ResponseField>

<ResponseField name="strike" type="number">
  Strike price for crypto markets
  
  Market resolves YES if spot price > strike
</ResponseField>

<ResponseField name="currentSpot" type="number">
  Current spot price of underlying asset
</ResponseField>

<ResponseField name="volume24h" type="number">
  24-hour trading volume (USDC)
</ResponseField>

<ResponseField name="liquidity" type="number">
  Total available liquidity (USDC)
  
  Aggregated across all venues
</ResponseField>

<ResponseField name="traders" type="number">
  Number of unique traders
</ResponseField>

<ResponseField name="settlement" type="object">
  Settlement information:
  - `source` - Oracle provider (Stork, Pyth, etc.)
  - `oraclePrice` - Final settlement price (null until settled)
  - `result` - Winning outcome (YES/NO, null until settled)
  - `settledAt` - Settlement timestamp (null until settled)
</ResponseField>

<ResponseField name="venues" type="object">
  Per-venue breakdown of liquidity and volume
</ResponseField>

### Example Request

<CodeGroup>

```bash cURL
curl "https://api.predifi.com/v1/markets?category=crypto&status=active&asset=BTC&limit=50" \
  -H "Authorization: Bearer YOUR_API_KEY"
```

```typescript TypeScript
const markets = await api.get('/v1/markets', {
  params: {
    category: 'crypto',
    status: 'active',
    asset: 'BTC',
    interval: '15M',
    sort: 'volume',
    order: 'desc',
    limit: 50
  }
});

console.log(`Found ${markets.data.markets.length} markets`);

markets.data.markets.forEach(market => {
  console.log(`\n${market.question}`);
  console.log(`  YES: ${(market.currentPrice.yes * 100).toFixed(1)}%`);
  console.log(`  NO: ${(market.currentPrice.no * 100).toFixed(1)}%`);
  console.log(`  Volume: $${market.volume24h.toLocaleString()}`);
  console.log(`  Liquidity: $${market.liquidity.toLocaleString()}`);
  console.log(`  Expires: ${new Date(market.expiresAt).toLocaleString()}`);
});
```

```python Python
response = requests.get(
    'https://api.predifi.com/v1/markets',
    headers={'Authorization': f'Bearer {api_key}'},
    params={
        'category': 'crypto',
        'status': 'active',
        'asset': 'BTC',
        'interval': '15M',
        'limit': 50
    }
)

markets = response.json()['data']['markets']

for market in markets:
    print(f"\n{market['question']}")
    print(f"  YES: {market['currentPrice']['yes']*100:.1f}%")
    print(f"  NO: {market['currentPrice']['no']*100:.1f}%")
    print(f"  Volume: ${market['volume24h']:,.0f}")
    print(f"  Liquidity: ${market['liquidity']:,.0f}")
```

</CodeGroup>

```typescript TypeScript
const market = await api.get('/v1/markets/BTC-15M-20250111-1430');

console.log(`Strike: $${m.strike.toLocaleString()}`);
console.log(`Current Spot: $${m.currentSpot.toLocaleString()}`);

console.log(`NO Price: ${(m.currentPrice.no * 100).toFixed(1)}%`);

console.log('\n=== Activity ===');
console.log(`24h Volume: $${m.volume24h.toLocaleString()}`);

console.log(`Oracle: ${m.settlement.source}`);

```

```python Python
response = requests.get(

print(f"NO Price: {m['currentPrice']['no']*100:.1f}%")

print("\n=== Activity ===")
print(f"24h Volume: ${m['volume24h']:,.0f}")
print(f"Liquidity: ${m['liquidity']:,.0f}")
print(f"Traders: {m['traders']}")
```

## Search Markets

Search markets by keywords.

<ParamField path="GET" type="method">
  /v1/markets/search
</ParamField>

### Request

<ParamField query="q" type="string" required>
  Search query
  
  Searches in market questions and descriptions
</ParamField>

<ParamField query="category" type="string">
  Filter by category
</ParamField>

<ParamField query="status" type="string">
  Filter by status
</ParamField>

<ParamField query="limit" type="number">
  Results limit (default: 20, max: 100)
</ParamField>

### Response

Same format as list markets, but filtered by search query.

### Example Request

<CodeGroup>

```bash cURL
curl "https://api.predifi.com/v1/markets/search?q=ethereum&category=crypto" \
  -H "Authorization: Bearer YOUR_API_KEY"
```

```typescript TypeScript
const results = await api.get('/v1/markets/search', {
  params: {
    q: 'ethereum',
    category: 'crypto',
    status: 'active'
  }
});

console.log(`Found ${results.data.markets.length} markets matching "ethereum"`);

results.data.markets.forEach(market => {
  console.log(`- ${market.question}`);
});
```

```python Python
response = requests.get(
    'https://api.predifi.com/v1/markets/search',
    headers={'Authorization': f'Bearer {api_key}'},
    params={'q': 'ethereum', 'category': 'crypto'}
)

markets = response.json()['data']['markets']
print(f"Found {len(markets)} markets")

for market in markets:
    print(f"- {market['question']}")
```

</CodeGroup>

## Get Price History

Get historical price data for a market.

<ParamField path="GET" type="method">
  /v1/markets/:id/prices
</ParamField>

### Request

<ParamField path="id" type="string" required>
  Market identifier
</ParamField>

<ParamField query="outcomeId" type="number">
  Outcome to query (0=YES, 1=NO)
  
  Omit to get both
</ParamField>

<ParamField query="interval" type="string">
  Price interval:
  - `1m` - 1 minute
  - `5m` - 5 minutes (default)
  - `15m` - 15 minutes
  - `1h` - 1 hour
</ParamField>

<ParamField query="from" type="number">
  Start timestamp (Unix ms)
  
  Default: market creation time
</ParamField>

<ParamField query="to" type="number">
  End timestamp (Unix ms)
  
  Default: current time
</ParamField>

### Response

```json
{
  "ok": true,
  "data": {
    "marketId": "BTC-15M-20250111-1430",
    "outcomeId": 0,
    "interval": "5m",
    "from": 1638345000000,
    "to": 1638360000000,
    "prices": [
      {
        "timestamp": 1638345000000,
        "price": 0.6200,
        "volume": 12345.67,
        "trades": 45
      },
      {
        "timestamp": 1638345300000,
        "price": 0.6350,
        "volume": 15678.90,
        "trades": 52
      }
      // ... more data points
    ]
  },
  "timestamp": 1638360000000
}
```

### Example Request

<CodeGroup>

```bash cURL
curl "https://api.predifi.com/v1/markets/BTC-15M-20250111-1430/prices?outcomeId=0&interval=5m" \
  -H "Authorization: Bearer YOUR_API_KEY"
```

```typescript TypeScript
const history = await api.get('/v1/markets/BTC-15M-20250111-1430/prices', {
  params: {
    outcomeId: 0,  // YES
    interval: '5m',
    from: Date.now() - (24 * 60 * 60 * 1000),  // Last 24 hours
    to: Date.now()
  }
});

console.log(`Got ${history.data.prices.length} price points`);

// Plot simple chart
history.data.prices.forEach(point => {
  const bar = 'â–ˆ'.repeat(Math.floor(point.price * 100));
  const date = new Date(point.timestamp).toLocaleTimeString();
  console.log(`${date}: ${bar} $${point.price.toFixed(4)}`);
});
```

```python Python
response = requests.get(
    'https://api.predifi.com/v1/markets/BTC-15M-20250111-1430/prices',
    headers={'Authorization': f'Bearer {api_key}'},
    params={
        'outcomeId': 0,  # YES
        'interval': '5m',
        'from': int((time.time() - 86400) * 1000),  # Last 24h
        'to': int(time.time() * 1000)
    }
)

prices = response.json()['data']['prices']

for point in prices:
    timestamp = datetime.fromtimestamp(point['timestamp']/1000)
    print(f"{timestamp}: ${point['price']:.4f} (vol: ${point['volume']:,.0f})")
```
