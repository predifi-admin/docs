## Circle CCTP

### What is CCTP?

**Circle's Cross-Chain Transfer Protocol** enables native USDC transfers between blockchains.

**How It Works:**
1. **Burn** - USDC burned on source chain
2. **Attest** - Circle verifies the burn
3. **Mint** - Native USDC minted on destination chain

**Key Benefits:**
- **Native USDC** - Not wrapped tokens
- **Zero fees** - No bridge protocol fees (only gas)
- **Secure** - Direct burn/mint by Circle
- **Simple** - No liquidity pools required

### Supported Routes

**Live Routes:**
- Arbitrum â†” Polygon, Base, Optimism
- Base â†” Polygon, Arbitrum, Optimism  
- Polygon â†” Arbitrum, Base, Optimism
- Optimism â†” Arbitrum, Base, Polygon

**Bridge Time:** ~15 minutes

**Cost Examples:**
- Arbitrum â†’ Base: ~$0.10 gas
- Base â†’ Polygon: ~$0.05 gas
- Polygon â†’ Arbitrum: ~$0.15 gas

### When We Use CCTP

**Primary use:** Moving USDC for trade execution

**Example:** User on Arbitrum wants to trade on Polymarket (Polygon)
1. User deposits USDC to Arbitrum LP Vault
2. Protocol bridges via CCTP: Arbitrum â†’ Polygon
3. Smart AA Wallet receives native USDC on Polygon
4. Execute trade on Polymarket
5. Profits returned to vault

## Wormhole

### What is Wormhole?

**Wormhole** is a generic cross-chain messaging protocol connecting 30+ blockchains.

**How It Works:**
1. **Lock** - Assets locked on source chain
2. **Verify** - Guardian network validates
3. **Mint** - Wrapped assets minted on destination
4. **Transfer** - User receives tokens

**Key Benefits:**
- **Generic** - Works with any token
- **Fast** - 1-2 minute finality
- **Secure** - 19-validator guardian network
- **Battle-tested** - $10B+ in volume

### When We Use Wormhole

**Primary use:** Bridging YES/NO position tokens back to user chains

**Example:** After trading on Polymarket (Polygon), bridge tokens to Arbitrum
1. User has YES tokens on Polygon
2. Lock YES tokens in Wormhole contract
3. Wormhole mints wrapped YES on Arbitrum
4. User receives tokens in wallet

**Also used for:**
- Non-CCTP chains
- Emergency fallback bridging
- Future non-USDC asset support
